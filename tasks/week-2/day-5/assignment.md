# Assignment

### Complete the expense tracker project

## Requirements

- Print out operations a user can carry out
- Ensure that the program keeps running even after reading, adding, updating or deleting.
- User can quit the program with 'q' and should be prompted y/n
- After updating or adding the user should be able to see the values they just added.
- At the end of the operation all interactions should be written to a file.
- Abstract into modules
